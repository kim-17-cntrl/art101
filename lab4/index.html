How to play Minesweeper

  1. Choose the size of the map
  2. Generate stuff on the map
    1. Choose the number of mines based on the map size
    2. Randomly put down mines on the map
    3. Generate tile numbers based on mine positions
  3. Display UI to the player
    1. Obscure tiles
    2. Include mines left on the map
    3. Include timer
  4. Check for mouse clicks on tiles
    1. If left click: reveal tile
      1. If mine, game over
      2. Else, display the number of lack or the number surrounding the tile
      3. If no number, reveal all touching tiles with no number
    2. If right-click: place marker
    3. If all non-mine tiles are revealed, the game wins
